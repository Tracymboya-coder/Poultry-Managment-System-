# 🐔 Poultry Management System

A smart, mobile-friendly web application designed to help farmers **track, analyze, and improve poultry farm performance**.

---

## 🚀 Features

### 📊 Farm Tracking

* Record daily data:

  * Number of chickens
  * Eggs collected
  * Feed used
  * Temperature
* View all records in a clean table

---

### 🧠 Smart Insights

* Total chickens count
* Average egg production
* Real-time updates after each entry

---

### ⚠️ Smart Alerts System

Automatically detects problems like:

* 🌡️ High temperature (heat stress)
* 🥚 Low egg production
* 🌾 Poor feed efficiency

---

### 🌦️ Weather Integration (Demo)

* Fetch current temperature
* Warns when conditions are too hot

---

### 🔍 Search Functionality

* Filter records by:

  * Day
  * Chickens
  * Eggs
  * Feed
  * Temperature

---

### 💾 Data Persistence

* Uses **localStorage**
* Data is saved even after page refresh

---

### 🎨 Modern UI/UX

* Clean dashboard layout
* Interactive hover effects
* Responsive (mobile-friendly)
* Font Awesome icons
